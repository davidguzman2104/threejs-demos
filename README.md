# 🌐 Three.js Virtual Environments Demo

Aplicación web desarrollada con **Three.js** que muestra distintos **ambientes virtuales interactivos en 3D** utilizando diferentes tipos de controles de cámara dentro de una misma aplicación.

El proyecto incluye **cuatro demos principales**:

- 🧱 **Minecraft Terrain**
- 🏙 **Map Controls (ciudad de edificios)**
- 🔷 **Orbit Controls (figuras geométricas)**
- 🎮 **Pointer Lock (entorno tipo videojuego FPS)**

---

# ⚙️ Tecnologías Utilizadas

- **Three.js** – Librería JavaScript para gráficos 3D.
- **WebGL** – Renderizado de gráficos 3D en el navegador.
- **JavaScript ES Modules**
- **HTML5**
- **CSS3**

# 🎮 Ambientes Virtuales

## 🧱 Minecraft Environment

Simula un terreno generado proceduralmente utilizando **Perlin Noise**.

Características:

- Generación automática de terreno.
- Control de cámara en **primera persona**.
- Texturas estilo Minecraft.
- Movimiento libre dentro del ambiente virtual.

Controles:
Mouse → mirar alrededor
Click izquierdo → avanzar
Click derecho → retroceder
## 🏙 Map Controls Environment

Genera una escena tipo **ciudad 3D** con edificios distribuidos aleatoriamente.

Características:

- 500 edificios generados proceduralmente.
- Colores personalizados: **azul, verde y morado**.
- Navegación similar a aplicaciones de mapas.

Controles:
Mouse → rotar escena
Scroll → zoom
Click + arrastrar → mover cámara

## 🔷 Orbit Controls Environment

Muestra múltiples **dodecaedros 3D** distribuidos en el espacio con colores aleatorios.

Características:

- 500 figuras geométricas.
- Colores: **azul, verde y morado**.
- Control de cámara orbital.

Controles:
Mouse → rotar cámara
Scroll → zoom
---

## 🎮 Pointer Lock Environment

Simula un entorno interactivo tipo **videojuego en primera persona (FPS)**.

Características:

- Movimiento libre en primera persona.
- Sistema de gravedad.
- Salto y colisiones.
- Objetos generados aleatoriamente en el entorno.

Controles:
Click → activar control
W A S D → movimiento
Mouse → mirar alrededor
Espacio → saltar

# 🔄 Navegación entre demos

Los ambientes pueden cambiarse utilizando el menú de navegación o las siguientes teclas:
1 → Minecraft
2 → Map
3 → Orbit
4 → Pointer Lock

# 📌 Descripción General

Este proyecto demuestra cómo crear **ambientes virtuales interactivos en el navegador** utilizando **Three.js**, combinando distintos tipos de controles de cámara, generación procedural de objetos y manipulación de escenas tridimensionales.

El objetivo es explorar distintas formas de navegación en entornos 3D dentro de aplicaciones web modernas.
